## API

### AsyncButton

| 参数 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| onClick |  点击按钮时的回调 | (event) => void \| Promise | - |
| delay | 延迟时间(毫秒) | number | - |
| ...others | 其他选项，同antd Button组件  | ButtonProps | - |
