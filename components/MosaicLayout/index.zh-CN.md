### 说明

常见于ide之类的面板，可以通过鼠标拖拽进行修改布局中的内容大小

## API

### AsyncButton

| 参数 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| children |  放入布局中的内容 | React.Node | - |
| splitWidth | 自定义拖拽条的宽度，单位：px | number | 8 |
| minWidth | 内容物的最小宽度，单位：百分比 | number | 4 |
